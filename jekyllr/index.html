Jekyllr - A simple Jekyll Admin









About

Jekyllr is a simple administration interface for your
jekyll blog. It is written entirely in javascript and is
completely static. Jekyllr was made possible by api calls

The workflow is simple. Head to Hastebin
and write your post in markdown along with the jekyll frontmatter.
Save the paste and copy the paste key from the url. eg: 

 If your url is: http://hastebin.com/ovigowiyex.vhdl then the 
	paste key is ovigowiyex. That is, the text between the /
	 and the random extension.
Give your post a title. This title will be included in the name of the .md
file. Enter your token decryption password and click sync post. Wait a few seconds
until you see the success signal.
